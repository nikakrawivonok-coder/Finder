# ANOMALY ARTIFACT GAME — CHECKPOINT V50.1

Поточна стабільна база для тесту: **V50.1 Stable Cleanup**.

## Що збережено з V41–V50

- Велика Зона / безкінечний забіг 100×100.
- Біоми, читання місцевості, привали, схрони, КПК загиблих.
- Фракції Зони, ефір, новини, фон секторів.
- Табори та безпечні місця.
- Легенда сталкера, чутки й впізнавання.
- Великий Викид, прогноз, шрами секторів, пошкодження таборів.
- Жива екосистема мутантів: сліди, території, міграції, полювання.
- Спадщина сталкерів: КПК, рюкзаки, мітки, схрони, останні записи.
- Міф про Виконавця бажань: уламки правди, брехня, знаки, поріг 7 доказів.
- Пробудження Зони: пам’ять гравця, розумні аномалії, пульс/воля Зони.

## Що зроблено у V50.1

- 10 окремих `ANOMALY_GAME_CHECKPOINT_V*.md` об’єднано в цей файл.
- 8 окремих `ROADMAP_AFTER_V*.md` об’єднано в `docs/ROADMAP.md`.

- `index.html` залишено головним single-file продуктом для GitHub Pages.

## Пріоритет тесту

1. Чи запускається базовий рейд.
2. Чи запускається Велика Зона.
3. Чи працюють кнопки V48–V50: спадщина, міф, пам’ять, аномалія, пульс.
4. Чи не зламався журнал КПК і остання подія під полем.
5. Чи не повторюються надто часто однакові повідомлення.

## Статус

Це не нова велика механіка, а **архітектурний checkpoint перед подальшим тестуванням**.
