# ANOMALY GAME CHECKPOINT V38 — Terrain Fear UI

## Головна ідея
Інтерфейс КПК більше не однаковий для всієї Зони. Кожен тип місцевості має власний візуальний режим, а критичний стан гравця вмикає окремий страх-інтерфейс.

## Додано
- `body.terrain-*` теми для локацій: Покинутий яр, Рудий ліс, Старий блокпост, Підземний тунель, Затоплена низина, Кладовище техніки, Пси-пляма, Стара наукова станція.
- Візуальна зміна кольорів, тону, фону, рамок, сигналів і клітин під місцевість.
- Короткий опис режиму місцевості в атмосферному блоці КПК.
- `nearDeathV38` страх-режим при HP ≤ 1 або радіації на межі.
- При провалі/смерті екран лишається у страх-режимі: червоне затемнення, пульсація, критичний біосигнал.

## Не змінено
- Бойова/рейдова математика.
- Генерація поля.
- Контакти Зони V37.
- Журнал КПК V36.

## Тест
1. Запустити кілька швидких рейдів — перевірити різні локації та різні кольори інтерфейсу.
2. Увійти у небезпечний стан HP 1 або радіація на межі — має з’явитися страх-режим.
3. Провалити рейд — результат має виглядати тривожно, не як звичайне меню.
4. Перевірити, що Контакти Зони і журнал працюють як у V37/V36.
